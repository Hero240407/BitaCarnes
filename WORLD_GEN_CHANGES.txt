GENERATION ALGORITHM:
====================

Mountains (groups of 3-24 tiles):
- Generate random center point
- Create circular cluster with Manhattan distance
- Fill density: 20-25% based on theme
- Radius: 2-4 tiles
- Only add if >= 3 tiles (minimum mountain size)

Lakes (groups of 7+ tiles):
- Generate random center point
- Create circular probability field
- Higher probability near center (natural shape)
- Fill density: 40-45% based on theme  
- Guaranteed >= 7 tiles or rejected and retried
- Avoid overlapping mountains

Villages:
- Generate like before
- Then immediately clear 5x5 protection zone around center
- Removes mountains and water (no more stuck villages!)

Paths:
- After all villages placed
- Find nearest villages and create A* routes between them
- Use stone paths for low-tech, asphalt for high-tech
- Automatically navigate around terrain obstacles

CUSTOMIZATION:
==============
To adjust generation, edit these parameters in modelos.py:

Mountain generation (line ~330):
- fill_rate = 0.20  (increase for denser mountains)
- raio = 2-4         (increase for larger clusters)

Water generation (line ~350):
- base_size = 10    (increase for bigger lakes)
- fill_rate = 0.40  (increase for more water)

Path generation (line ~385):
- tech_avg >= 2.5   (lower for more asphalt roads)

Zone clearing (line ~418):
- raio_protecao = 2 (increase for bigger safe zones)
