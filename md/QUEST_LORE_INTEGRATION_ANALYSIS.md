## 1. CURRENT QUEST GENERATION SYSTEM

### Quest Data Structure (`jogo/quests.py`)
Located in: [jogo/quests.py](jogo/quests.py)

**Quest Class:** Core quest object with these properties:
- `id`: Unique identifier
- `nome` / `descricao`: Quest title and description
- `tipo`: Quest type enum (COLETA, ENTREGA, DERROTA, DESCOBERTA, MELHORIA, CRIACAO, RELACIONAMENTO, EVENTO, MINIGAME)
- `status`: Current state (DISPONIVEL, ATIVA, COMPLETA, FALHADA, ABANDONADA)
- `npc_giver` / `npc_target`: NPC relations
- `objetivo_quantidade`: Progress tracking
- `dificuldade`: 1-5 difficulty level
- `recompensa`: Gold, EXP, items, unlocks, NPC relation increases
- `repetiavel`: Whether quest can be repeated
- `diaria` / `semanal`: Daily/weekly flags

### Quest Generation Methods

**Current Quest Initialization** ([jogo/quests.py](jogo/quests.py#L154-L201)):
1. **_criar_quest_colheita()** - Harvest 5 items, given by "Raphael"
2. **_criar_quest_encontro_npc()** - Meet 3 NPCs
3. **_criar_quest_melhoria_ferramenta()** - Upgrade a tool (requires 100 gold)
4. **_criar_quest_festival()** - Participate in festival

**Random Quest Generation** ([jogo/quests.py](jogo/quests.py#L253-L299)):
```python
gerar_quest_randomica() → Quest
```
- Generates random quest from limited pool (COLETA, ENTREGA, MELHORIA)
- Uses `_gerar_nome_quest()` and `_gerar_descricao_quest()`
- **PROBLEM:** Descriptions are generic:
  - "Colete os itens necessários para completar essa tarefa"
  - "Entregue os itens para a pessoa correta"
  - "Melhore seus equipamentos e habilidades"

**Raphael Quest Generation** ([jogo/modelos.py](jogo/modelos.py#L665-L681)):
```python
mundo.gerar_quest_raphael() → dict
```
- Only updates `quests_ativas` list with hardcoded choices:
  - "Investigue rumores de guerra entre dois imperios."
  - "Proteja uma aldeia durante um festival religioso."
  - "Busque conhecimento antigo em uma biblioteca esquecida."
  - "Converse com um mestre arcano e aprenda uma magia nova."

### Key Limitation
**Quests are NOT currently generated by AI.** They're:
- ✗ Not contextualized to world lore
- ✗ Not connected to world narrative
- ✗ Not affected by world_lore values
- ✗ Not personalized to player lore/origins
- ✗ Not dynamically created based on Raphael's personality
- ✗ Hardcoded or minimally randomized

## 2. WORLD LORE SYSTEM

### World Lore Generation
Located in: [jogo/sociedade.py](jogo/sociedade.py#L208-L280)

**Function:** `gerar_lore_mundo(nome_humano: str) → dict`

**World Lore Structure:**
```python
{
    "eixo_historico": string,      # Historical axis
    "condicao_raphael": string,    # Raphael's current state/role
    "era_inicial": int,            # Starting era (1500, 1600, 1700, 2200, 3000, 5000)
    "profecia": string,            # Prophecy involving player name
    "legenda": string,             # World legend
    "tema_terra": string,          # Geographic theme
    "conflito_principal": string,  # Main conflict
    "lugar_mstico": {
        "nome": string,            # Mystical place name
        "descricao": string,       # Description
    }
}
```

**World Lore Examples:**

Historical Axes:
- "O mundo foi rasgado por uma guerra entre ceu e abismo"
- "Uma chuva de mana alterou a historia dos reinos"
- "Um isekai aconteceu, trazendo herois de outro mundo"

Raphael Conditions:
- "Raphael esta forte, mas politicamente contido pela Igreja do Verbo"
- "Raphael foi selado em um cristal antigo horas antes de seu retorno"
- "Raphael está dividido entre misericórdia e justiça"

Prophecies (always refer to player):
- "{player_name} pode alterar o destino de imperios"
- "{player_name} quebrará a sétima selagem e libertará o caos"
- "{player_name} será a ponte entre o mundo antigo e novo"

Themes:
- terra_mutante, reino_fragmentado, ruinas_mysticas
- isekai_convergente, era_draconica, apocalipse_magico

### World Lore Storage
- **Created:** In `Mundo.__init__()` via `gerar_lore_mundo(nome_humano)`
- **Stored:** `mundo.world_lore` (dict)
- **Persisted:** Saved in `lore_quests.json`
- **Used:** Affects terrain generation theme selection ([jogo/modelos.py](jogo/modelos.py#L317-L318))

### Storage Location
```
saves/{save_name}/lore_quests.json
```

### Key Observation
**World lore is barely used!**
- ✗ Only terrain theme affects world generation
- ✗ Prophecy is never checked
- ✗ Raphael's condition is never consulted
- ✗ Main conflict is not driving gameplay
- ✗ Legends are informational only

## 6. INTEGRATION OPPORTUNITIES & RECOMMENDATIONS

### 6.1 QUEST GENERATION IMPROVEMENTS

**Current Limitation:** Only 4 hardcoded quests + generic random quests

**Recommendation:** Create `gerar_quest_dinamica()` that:

```python
def gerar_quest_dynamica(mundo, raphael) → Quest:
    # 1. Extract world context
    world_context = mundo.world_lore
    player_lore = mundo.perfil_jogador
    
    # 2. Build context prompt
    prompt = f"""
    Você é Raphael gerando uma quest contextualizada.
    
    MUNDO:
    - Profecia: {world_context['profecia']}
    - Conflito Principal: {world_context['conflito_principal']}
    - Legenda: {world_context['legenda']}
    
    JOGADOR:
    - Nome: {player_lore['nome']}
    - Origem: {player_lore['origem']}
    - Legado: {player_lore['legado']}
    - Motivação: {player_lore['motivacao']}
    
    Gere 1 quest que:
    - Use elementos da profecia
    - Conecte ao conflito principal
    - Considere a origem/motivação do jogador
    - Seja único e narrativo
    
    JSON: {{
        "titulo": string,
        "descricao": string,
        "tipo": "coleta|entrega|derrota|...",
        "objetivo_quantidade": int,
        "dificuldade": 1-5,
        "recompensa_ouro": int,
        "recompensa_exp": int,
        "npc_giver": string,
        "ai_generated": true
    }}
    """
    # 3. Call Ollama
    resposta = chamar_ollama_pesado(prompt)
    # 4. Parse and create Quest
    return quest_from_dict(json.loads(resposta))
```
